# PDF Tool Suite

A privacy-first PDF toolkit for merging, splitting, compressing, and protecting PDFs directly in the browser.

## Features

- 100% client-side processing with `pdf-lib`
- Merge multiple PDFs with file reordering
- Split PDFs by page ranges (example: `1-3, 5, 7-10`)
- Basic compression (metadata cleanup + form flattening)
- Password protection workflow
- Drag-and-drop uploads with status + file size feedback
- Dark mode and mobile responsive UI

## Privacy

Your files never leave your browser.

## Canonical URL

`https://devhandbook.io/pdf-tools/`

## Run locally

Open `pdf-tools/index.html` in a modern browser.
