## Quick Start

Simply open `index.html` in any modern browser. No build step, no server required.

```bash
# Clone or download
cd jwt-debugger

# Option 1: Open directly
open index.html

# Option 2: Simple HTTP server (for testing)
python3 -m http.server 8000
# Then visit http://localhost:8000
```
